

# Pronunciation preferences

Speech synthesizers may mispronounce certain strings like "LOL" or "a11y".  
This page lets you configure Orca’s pronunciation dictionary with application-specific overrides.

## Adding a New Dictionary Entry

1. Press **New Entry** (`Alt+N`)
2. Type the original string and press `Return`
3. Move to the **Replacement String** column and press `Return`
4. Type the replacement and press `Return` again

## Editing an Existing Entry

1. Select the desired entry
2. Press `Return` to edit
3. Make changes and press `Return` to save

## Deleting an Entry

1. Select the entry
2. Press the **Delete** button or use `Alt+D`
