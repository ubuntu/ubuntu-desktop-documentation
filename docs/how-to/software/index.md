# Install or remove software

By default, many useful programs are already installed when you set up Ubuntu on your computer. However, you may need a particular piece of software for a purpose not served by the default applications.

You can install applications and other software that comes from Ubuntu or third-party sources. You can also update or uninstall software.

If you'd like to get some background information on what's happening when you install software, read {ref}`snap-and-deb-packages`.
