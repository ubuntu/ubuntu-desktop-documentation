# Remove an application

An application is software that has a graphical user interface (GUI). You can use *App Center* to remove applications that you no longer use.

Applications are available in two formats: snap packages ("snaps") and Debian packages ("Debs"). Some applications are available in both formats. In such a case, the snap will be listed first in App Center.

:::{note}
You can also use App Center to remove software that doesn't have any GUI.
:::

To remove an application:

1. Click the App Center icon in the Dock, or search for *Software* in the Activities search bar.

1. When App Center opens, click the {guilabel}`Manage` button in the side bar.

1. Find the application that you want to remove by looking through the list of installed applications, or by using the search bar.

1. Select the application and click {guilabel}`Uninstall`.

1. Enter your password to confirm that you want to remove the application.

:::{note}
Some applications depend on other applications to work properly. If you try to remove an application that is needed by another application, both will be removed. App Center will ask you to confirm whether you want this to happen before the applications are removed.
:::
